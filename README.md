## Features
-Search weather by city name

-Current temperature display

-"Feels like" temperature

-Humidity Infromation

-Wind Speed

-Atmospheric Pressure

-Visibility data

-5-day weather forecast

-Dynamic weather icons based on conditions 

-Loading screen while fetching weather data

-Error handling for invalid city names

-Search using Enter Key

-LocalStorage support for saving the last searched city

## Technologies Used
-HTML

-CSS

-JavaScript

-Fetch API

-Async/Await

-LocalStorage

-Weather API
